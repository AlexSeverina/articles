\abstract{Context-free grammar, string-matching, Valiant's algorithm, secondary structure, parsing.
\\[17pt]
{\bf Abstract.} Some kinds of the string-matching problem can be reduced to parsing---verification if some subsequence can be derived in the given grammar. Bioinformatics requires working with a large amount of data, so it is necessary to improve the existing parsing techniques. The most asymptotically efficient parsing algorithm that can be applied to any context-free grammar is a matrix-based algorithm proposed by Valiant. This paper presents Valiantâ€™s algorithm modification, which provides the possibility to divide the parsing table into successively computed layers of disjoint submatrices where each submatrix of the layer can be processed independently. Moreover the modified version decreases a large amount of excessive computations and accelerates the substrings searching. }
